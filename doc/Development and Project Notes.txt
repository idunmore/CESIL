CESIL (Computer Education in Schools Intruction Lanauge) Interpreter

CESILPlus - (Optional) Updates to CESIL Language

 * Default "STRICT" CESIL interpretaion w/ OPTIONAL "PLUS" mode:

   * Adds a STACK with PUSH/POP commands.
   * Adds SUBROUTINE support (with conditional jumps)

 * Supports "TEXT" (free-er form) and "CARD" (fixed column) source code files.

 * Adds "Debugging" - effectively two levels of optional state output.

 * More robust and complete handling of errors.

 * Proper Command Line Interface (CLI)

 * PEP-8 Compliant

 * Simpler Code - No "Clever Tricks"

 DEBUG MODES:

  0 = No Debugging
  1 = Summary output
  2 = Summary output w/ stop
  3 = Verbose output
  4 = Verbose output w/ stop
